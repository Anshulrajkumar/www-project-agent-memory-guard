# Content Engine: Talks → Evergreen Surface Area

Each conference talk should spawn multiple discoverable assets. The talk is the hard part — this plan multiplies each one into months of organic reach.

---

## The Formula

One talk produces:

1. **60-second attack-demo clip** → LinkedIn + YouTube Shorts
2. **Written companion post** → Dev.to + personal blog
3. **Slides on SpeakerDeck** → SEO + embed in blog posts
4. **Thread/carousel** → LinkedIn + X
5. **Newsletter submission** → tl;dr sec, TLDR AI

## Short-Form Video Strategy

### Format (60 seconds)
```
0-5s:   Hook — "Watch this agent get permanently hijacked"
5-20s:  Setup — show normal agent behavior
20-35s: Attack — inject poisoned memory
35-45s: Persistence — context reset, attack persists
45-55s: Defense — AMG blocks the write
55-60s: CTA — "pip install agent-memory-guard"
```

### Distribution
| Platform | Format | Frequency |
|----------|--------|-----------|
| YouTube Shorts | Vertical 9:16 | 2x/month |
| LinkedIn Video | Square 1:1 | 2x/month |
| X/Twitter | Horizontal 16:9 | 1x/month |
